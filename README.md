\# 🎉 Giri Event Registration Platform



A full-stack web application for discovering and registering for events with location-based recommendations. Built with Node.js, Express, MongoDB, and vanilla JavaScript.



\## ✨ Features



\- 🔐 \*\*User Authentication\*\* - Secure registration and login with JWT tokens

\- 📍 \*\*Location-Based Events\*\* - Auto-detects user location to show nearby events

\- 🎟️ \*\*Event Registration\*\* - One-click registration for events

\- ⭐ \*\*Reviews \& Ratings\*\* - Users can rate and review events

\- 🎨 \*\*Modern UI\*\* - Beautiful, responsive design with smooth animations

\- 📱 \*\*Mobile Friendly\*\* - Works on all devices



\## 🛠️ Tech Stack



\### Backend

\- \*\*Node.js\*\* - JavaScript runtime

\- \*\*Express.js\*\* - Web framework

\- \*\*MongoDB\*\* - NoSQL database

\- \*\*Mongoose\*\* - ODM for MongoDB

\- \*\*JWT\*\* - Authentication tokens

\- \*\*bcryptjs\*\* - Password encryption



\### Frontend

\- \*\*HTML5\*\* - Structure

\- \*\*CSS3\*\* - Styling with animations

\- \*\*JavaScript\*\* - Interactivity

\- \*\*Font Awesome\*\* - Icons

\- \*\*Google Fonts\*\* - Typography
